# cpu

主要由两部分构成:控制单元&算术逻辑单元(ALU)

控制单元:从内存中提取指令并解码执行  
ALU:处理算数&逻辑运算

按功能分成四部分

寄存器:中央处理器的组成部分,用来暂存指令,数据,地址.可以看作是内存的一种.一个cpu内部会有20-100个寄存器

控制器:负责把内存上的指令,数据读入寄存器,根据指令结果控制计算机  

运算器:负责运算从内存读入寄存器的数据  

时钟:负责发出CPU开始计时的时钟信号


累加寄存器:存储运行和运算后的数据

标志寄存器:反应处理器的状态和运算结果的某些特征以及控制指令的执行

程序计数器:程序计数器存放下一条指令所在单元的地址

基址寄存器:存储数据内存的起始位置

变址寄存器:存储基址寄存器的相对地址

通用寄存器:存储任意数据

指令寄存器:储存正在被运行的指令，CPU内部使用，无法对该寄存器进行读写

栈寄存器:存储栈区域的起始位置

程序计数器,累加寄存器,标志寄存器,指令寄存器,栈寄存器只有一个

# 内存
主要分为三种存储器:RAM,ROM,Cache

RAM:随机存储器,可以读取写入,机器关闭时,内存中的信息会丢失

ROM:只读存储器,只能读取,断电时数据不会丢!

Cache:解决主存与CPU速度不匹配,分为三级缓存,比内存读写更快

# 机器数
机器数:二进制形式,最高位存放符号(正0负1)  
真值:机器数对应真正的数值(不是二进制)

原码:符号位+真值的绝对值  
反码:整数的反码是本身,负数的反码是符号位不变,其他位取反(反码可以让减法变成加法)  
补码:正数的补码也是本身,负数补码是符号位不变,其余各位取反再+1  
移码:只有整数,没有小数,且移码与补码数值位相同,符号位不同

模的定义:x mod y = x - y*[x/y]

# 存储器性能指标
存储容量:存储字数*字长  
单位成本:每位价格=总成本/总容量  
存储速度:数据传输率=数据的宽度/存储周期

存取时间(Ta):分为读出时间和写入时间,指启动一次存储器到完成的时间  
存取周期(Tm):指存储器进行完整的读写操作所需的全部时间  
主存带宽(Bm):数据传输率,每秒主存进出信息的最大数量,单位字节/秒(B/s)位/秒(b/s)
